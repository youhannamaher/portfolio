I developed an AI-powered HR system prototype to demonstrate how recruitment and HR processes can be fully automated using Microsoft Power Platform. The solution includes a complete Applicant Tracking System (ATS) that allows recruiters to create job postings, collect candidate applications through online forms, and manage the recruitment pipeline in a structured and efficient way.

Candidate applications and CVs are automatically processed using Power Automate and AI Builder, which extract key information and relevant skills from uploaded CVs and store them in a structured format. Recruiters can then review candidates through a PowerApps interface, filter applicants by criteria such as degree level or keywords, and prioritize candidates based on AI-driven relevance. The system also allows viewing CVs directly within the application and managing candidate progression through different recruitment stages.

The platform includes automated communication workflows, sending personalized HTML emails to candidates, including interview invitations for selected applicants and detailed feedback for rejected candidates. In addition, a payroll module is integrated to track working hours, calculate compensation, and automate salary notifications. Built using Power Apps, Power Automate, AI Builder, and SharePoint, this project demonstrates how HR operations can be transformed into a scalable, automated, and data-driven system.
